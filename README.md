# MM8
Fully Metal Frame for Anet A8
<img width="880" height="677" alt="imagen" src="https://github.com/user-attachments/assets/bd09f32f-9343-4938-a566-f2507773a285" />

The file is a Parasolid file. Feel free to make pull requests
MM8 is a re-design of the A8 3D printer, featuring:
a compartment for Filament Spool with nice looking door, 
hided PSU and main board,
sheet metal parts,
1 1/4 " Square Tube Frame,
it can be enclosed with acrylic or glass,
motors and other enclosures can be 3d-printed,
the wiring can be hided,
almost all the printable parts can be fixed with M3 bolts.
